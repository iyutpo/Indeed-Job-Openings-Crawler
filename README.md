# Indeed-Job-Openings-Crawler

## Data Available:
* 1. Job title
* 2. Company name
* 3. Location
* 4. Salary
* 5. Summary (Job Description)
* 6. Sublink (Detailed Job Description)
* 7. Full-Summary (Full Job Description)

## Note: You can change "keyword" and "the number of jobs" before running this crawler.
